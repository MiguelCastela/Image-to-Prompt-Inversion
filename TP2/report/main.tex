\begin{abstract}
This report presents a comprehensive methodology for the image-to-prompt inversion problem under a fixed latent diffusion generator. The objective is to recover discrete text prompts that accurately re-render six target images using the Latent Consistency Model (LCM) Dreamshaper v7. Because the generation setup---including the random seed, inference steps, and guidance scale---is strictly fixed, the challenge shifts from generating fluent text to an optimization problem over the discrete tokenizer vocabulary. We propose a hybrid pipeline that combines Vision-Language Model (VLM) captioning for semantic initialization, Large Language Model (LLM) driven evolutionary sampling for structural refinement, and gradient-based discrete optimization (e.g., PEZ) for final pixel-level alignment. Performance is strictly evaluated on image-side metrics: CLIP image-image similarity, LPIPS perceptual distance, and Pixel Mean Squared Error (MSE).
\end{abstract}

Prompt inversion involves recovering the textual conditioning that generated a specific synthetic image. While traditional image captioning focuses on natural language fluency, latent diffusion models are highly sensitive to specific aesthetic tags and structural modifiers. This project addresses the challenge of recovering prompts that reproduce exact target images under a fixed text-to-image pipeline. Building upon previous explorations of generative modeling, this work treats prompt inversion not as a linguistic translation task, but as an optimization problem where the loss function is computed directly in the pixel and perceptual domains. 

\section{Problem Description}
The core task is to invert six target images generated by \texttt{SimianLuo/LCM\_Dreamshaper\_v7}. The generation environment is entirely deterministic: the model checkpoint, scheduler, 8 inference steps, 8.0 guidance scale, and 768x768 resolution are constant. Crucially, the random seed for each target is known and parsed directly from the filename. 

Because the generation pipeline is locked, the textual prompt is the sole optimizable variable. The evaluation explicitly ignores the grammatical correctness of the prompt; rather, success is defined by how closely the rendered pixels match the target pixels. Consequently, methods must prioritize specific "SD-speak" modifiers (e.g., "highly detailed", "sharp focus") over coherent prose.

\section{Methodology and Approach}
To navigate the vast and non-differentiable discrete prompt space, we implemented a multi-stage approach combining semantic retrieval and mathematical optimization.

\subsection{Phase 1: Semantic Initialization via Vision-Language Models}
To provide a warm start, target images are processed through CLIP-Interrogator and BLIP-2. BLIP-2 provides an accurate syntactic description of the subjects, while the CLIP-Interrogator extracts the necessary stable diffusion aesthetic tags and artist references.

\subsection{Phase 2: LLM-Based Iterative Refinement}
The initial captions are expanded into a broad set of candidate prompts using an LLM. By rendering these candidates and scoring them against the target, we establish an evolutionary loop. The LLM acts as an optimizer, mutating the highest-scoring prompts by altering spatial descriptions and keyword weights based on image-side feedback.

\subsection{Phase 3: Gradient-Based Discrete Optimization}
For top candidates that plateau in pixel MSE, we employ gradient-based discrete optimization strategies such as Hard Prompts Made Easy (PEZ). Initialized with our best LLM-refined prompts, PEZ calculates gradients across the continuous embedding space and projects them back onto the discrete tokenizer vocabulary, driving the LPIPS and MSE metrics toward zero.

\section{Experimental Setup}
All experiments were conducted using the provided Colab/VS Code environment. The rendering pipeline uses the \texttt{SimianLuo/LCM\_Dreamshaper\_v7} model. 
Evaluation is performed using three mandatory metrics:
\begin{itemize}
    \item \textbf{CLIP Image-Image Similarity:} Measures the cosine similarity between the target and generated image embeddings using \texttt{openai/clip-vit-large-patch14}.
    \item \textbf{LPIPS:} Evaluates the learned perceptual feature distance.
    \item \textbf{Pixel MSE:} Computes the strict mean squared error in pixel space.
\end{itemize}

\section{Conclusion}
This report presented a hybrid methodology for prompt inversion. By treating the generator as a fixed mathematical function and shifting the search space from natural language to localized token optimization, we successfully recovered prompts that align tightly with the target distributions. Future iterations could explore more aggressive soft-prompt projections prior to discrete snapping to further minimize structural differences.
